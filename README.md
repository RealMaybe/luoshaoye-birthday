# 洛少爷生日倒计时

## 介绍

洛少爷生日倒计时手机版软件，前台基于H5+App进行架构，使用原生JS+jQuery来完成页面相关效果。

-----

## 软件架构

现版本包含页面：

- **首页：** 倒计时计时器、背景轮播效果
- **信息：** 官方公告
- **我的：** 个人用户登录、日历

-----

## 安装教程

**网页版：** 暂时不提供相关支持
**发行版：** 下载*.apk程序包，在安卓系统内安装运行
